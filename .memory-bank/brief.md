# Project Brief: Wildtide

## One-Liner
**Wildtide** — An indirect auto-city builder where you govern through policy, not placement. *WorldBox* meets *Frostpunk* in a *Terra Nil* aesthetic.

## Key Facts

| Field | Value |
|-------|-------|
| **Genre** | Auto-City Builder / Indirect Management / Strategy |
| **Engine** | Godot 4.6 (stable, pinned) |
| **Language** | GDScript only |
| **Platform** | PC — Linux Native (primary) |
| **Target HW** | AMD integrated GPU (Steam Deck tier), 60fps |
| **Campaign** | ~2 hours (16 cycles × ~8 min/cycle) |
| **Dev Team** | Solo developer |

## Elevator Pitch

In a post-apocalyptic world scarred by failed wormhole experiments, you guide a city's survival not by placing buildings, but by setting policies, approving faction quests, and managing metrics. AI citizens build and evolve autonomously based on your decisions. Every few minutes, "The Wave" — monsters from dimensional Rifts — tests everything you've built. Lead your civilization down the path of Science or Magic to seal (or tame) the Rifts for good.
